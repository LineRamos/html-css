Exemplo de Cores - Método de escolha "automática" para RGB

  • Nesse método, para a cor aparecer de forma automática (sem ter que decorar os parâmetros da cor, seja em RGB, HSL ou hexadecimal).
  • Para isso, deve digitar a cor escolhida, posicionar o cursor sobre o nome da cor. Vai aparecer uma janela com todo o espectro daquela cor. Ao correr o cursor sobre as cores, vai alterar automaticamente para o parâmetro RGB.
  • Caso deseje mudar para HSL ou Hexadecimal, é só clicar no título da caixa que vai alterar de acordo com a preferência.
  • Para adicionar uma transparência - entre a caixa de cores e uma barra à direita de cores, tem uma barra. Essa barra define se terá transparência na cor. Caso deseje, clique em cima e percorra para definir o grau de transparência. Isso fará com que o código final da cor seja alterado (automaticamente), devido a parte da transparência.
